## Decisions

**D-T4-1 — Where the amendment was recorded.** The task required recording that DEC-6 was
amended; the map's §1 replacement row contains no amendment marker, and the map required that
row be used verbatim. Rather than modify the mandated row, I added a separate note immediately
after the §6.1 decision table and before §6.2. It states that DEC-6 was amended in place, not
renumbered and not superseded by a new ID, names what changed, and records that the address
portion and all other DEC-1…20 entries are unamended. This satisfies both instructions without
altering a single character of the map-specified text. The note contains no forbidden token, no
`$`-amount, no rail name and no vendor name, so it moves no counter.

**D-T4-2 — Map item [1] applied verbatim despite an internal inconsistency.** The map's
replacement for `01:256` reads `ovog/etsgiin ner/ner` — "etsgiin ner" with a **space**, where
every other item uses the underscored field token `etsgiin_ner`. The rule was to use exact
replacements verbatim, so I applied it as written. It is defensible as prose (the surrounding
text is a feature description, not a schema listing) and it moves no counter either way. Flagged
under Follow-ups so a reviewer can decide whether to normalise it.

**D-T4-3 — Two out-of-map name-shape occurrences left untouched, not fixed.** See Blockers.
The instruction was to stop and report rather than improvise when the map does not cover
something. I did not edit them.

**D-T4-4 — `04:109` draft-comparison columns preserved byte-identical.** They read
"first/last, no address" ×2. That wording carries no forbidden token, is a historical record of
what the superseded drafts said, and the map's prose rule requires it be left exactly as-is. I
changed only the bolded final resolution column. I did not expand, correct, or annotate the
historical columns. `[VERIFIED: diff of 04:109 touches only the final column]`

**D-T4-5 — Scope held to names.** No currency, rail, vendor, address-model, threshold or
timezone text was touched. The address clauses inside DEC-6 were carried through verbatim as
part of the map's replacement row. All three DRIFT counters sitting exactly at baseline is
corroborating evidence that scope did not widen. `[VERIFIED: verifier output, all three = baseline]`
